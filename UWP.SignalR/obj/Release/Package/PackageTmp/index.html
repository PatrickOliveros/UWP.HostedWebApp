    About This Project

    This project is a proof of concept where ASP.NET SignalR can be used to perform various responses to actions in certain line of business (LOB) applications using a Hosted Web Application with Windows 10. More information can be found in this link.

    Grayscale is a free Bootstrap 3 theme created by Start Bootstrap. The theme is open source, and you can use it for any purpose, personal or commercial.

    Application Toast

    This button will trigger a web notification in a browser, otherwise a toast if inside a hosted application. The toast simulates as if a normal business logic takes place. The toast will display a fictional business object (request number) and the current status called from an ASP.NET Web API endpoint.

    Activation of this toast will just append the business object information in the status box.
